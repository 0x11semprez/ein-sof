### ein-sof `אין סוף` — algorithm laboratory in pure c
└⟁⟁⟁┘

### ⟐ directives
  ⚙ zero dependency — every structure forged from scratch<br>
  ⚙ memory-first — manual control over every byte<br>
  ⚙ complexity-aware — annotated & benchmarked on silicon<br>
  ⚙ battle-tested — 247+ unit tests, valgrind clean<br>

### ⟐ modules
  ⚙ data structures — vector, list, stack, queue, deque, bst, avl, rbt, b-tree, trie, heap, hashmap, hashset, bloom filter, graph<br>
  ⚙ algorithms — 10 sorts, binary/interpolation search, bfs, dfs, dijkstra, a*, mst, tarjan, topo sort<br>
  ⚙ dynamic programming — knapsack, lcs, edit distance, lis, coin change, matrix chain<br>
  ⚙ strings — kmp, rabin-karp, boyer-moore, aho-corasick, z-algorithm, manacher<br>
  ⚙ mathematics — bigint, modular arithmetic, primes, matrix ops, strassen<br>
  ⚙ core — arena allocator, pool allocator, debug allocator<br>

### ⟐ stack
  ⚙ c99 · gcc/clang<br>
  ⚙ make · criterion · valgrind<br>
  ⚙ zero external dependencies<br>

### ⟐ quickstart
```bash
git clone https://github.com/yourusername/ein-sof.git && cd ein-sof
make              # build
make test         # 247 tests
make bench        # benchmarks
make valgrind     # memory audit
```

### ⟐ api convention
  ⚙ `es_<module>_<action>()` · `t_` prefix · snake_case everywhere<br>
